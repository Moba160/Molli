# Tab Strecke
Visualisierung der Strecke mit allen Bahnhöfen.


In der Titelleiste kann die Zeit gewählt werden, um die aktuellen Züge auf der Strecke zu sehen.
Die Züge werden auf der Strecke farblich markiert. Orange für die Richtung Bad Doberan - Kühlungsborn West und blau für die Richtung Kühlungsborn West - Bad Doberan.


Jeder aktuelle Zuglauf wird in einer diesem Farbschema folgenden Tabelle dargestellt. Der nächste Halt ist hervorgehoben, bereits abgefahrene Halte sind ausgegraut.


Die Zugpositionen werden gemäß der in der Titelleiste gewählten Zeit aus dem Fahrplan abgeleitet. Mit dem Schieberegler kann die Zeit beliebig (von der aktuellen Uhrzeit abweichend) eingestellt werden. Entsprechend wandern die Zugpositionen.


Folgende Tastaturkürzel existieren:
* linker Pfeil: eine Minute zurück
* rechter Pfeil: eine Minute vor    
* Pfeil auf: an den Anfang springen
* Pfeil ab: an das Ende springen
* Enter: auf die aktuelle Uhrzeit springen

Wenn der Standort freigegeben wird, wird er durch einen pulsierenden Punkt auf dem Display hervorgehoben. Das macht nur Sinn - und dafür ist es gedacht - wenn man sich auf einem Mobilgerät in der Nähe der Strecke befindet, um bspw. für das Fotografieren den nächsten Zug zu sehen.
